Introduction


Sur cette page, vous allez découvrir acteur fondamental du développement web, le couple HTML+CSS (Hyper Text Markup Langage et Cascading Style Sheets) Grâce au HTML vous allez pouvoir, dans votre navigateur (Firefox, Chrome, Opera,....), afficher du texte, afficher des images, proposer des hyperliens (liens vers d'autres pages web), afficher des formulaires et même maintenant afficher des vidéos (grâce à la dernière version du HTML, l'HTML5). Il est souvent utilisé conjointement avec des langages de programmation (JavaScript) et des formats de présentation (CSS).

Pourquoi apprendre HTML/CSS ?


  • Le CSS permet à l’utilisateur de personnaliser une page web, de nombreux sites web permettent à l’utilisateur de changer la mise en page d’un site sans modifier le contenu.
  • Les pages web se chargent plus rapidement et utilisent moins de bande passante car la feuille de style (téléchargée une seule fois puis stockée dans la mémoire cache) simplifies le code HTML ce qui permet de consommer moins de bande passante lors du chargement de la page.
  • Le CSS est compatible avec tous les navigateurs, les sites web qui emploient correctement le CSS ont une apparence similaire quelque soit le navigateur utilisé.
  • Le CSS facilite le positionnement dans les moteurs de recherche, le contenu est fourni en premier pour qu’il soit plus facilement « capturable » par les obots d’indexation.

Les balises html de base


                    
    
                        
  • <h1>Permet d'afficher un titre parmis 6 tailles de texte différents du h1 au h6 (h1 étant la plus grosse taille)</h1>
    • 
                        
  • <P>Permet d'afficher du texte, on l'utilise pour les paragraphes généralement</P>
    • 
                        
  • <a>Permet de creer un lien, cela peut être un lien vers un élément,sur la page ou vers une autre page</a>
    • 
                        
  • <img>Permet d'afficher une image, de lui attribuer un nom et une taille, un lien permet de choisir une image stockée en local ou bien d'indiquer l'url d'une image stockée en ligne</img>
    • 
                        
  • <ul>Permet d'afficher une liste sans ordre de tri (il est possible d'afficher une liste ordonnée avec la balise <ol>)</ul>
    • 
                        
  • <li>Permet d'afficher un élément d'une liste avec bullet point (par déffaut)</li>
    •

Tips & bonnes pratiques

Voici quelques astuces qui pourront vous aider sur vos projets en html et css:


  • Pensez en premier lieux à lier votre feuille css à votre page html à l'aide de la balise link (à placer dans la balise <head> de votre index.html)
  • Le site https://www.w3schools.com/ qui est une bible pour retrouver toutes les fonctions de html et de css et permets de tester le code en ligne directement.
  • L'editeur de code "Visual studio code" et son pulgin "live server" vous permettent d'afficher la page hmtl pour visualiser l'avancée du projet.
  • Le racourci ctrl+/ qui permet d'ajouter des commentaires (qui n'apparaissent que dans le code) vous permetant d'avoir un code plus lisible
  • Essayez d'organiser votre css dans le même ordre que votre html (de l'élément le plus en haut de la page jusqu'au plus bas)
